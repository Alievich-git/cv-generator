CV Generator dashboard

Do the AI conversation in your IDE. This page shows the live resume preview, ATS score, keywords, report, and agent notes for the latest run.

Start from your IDE

In any agentic IDE that can read this repo's instructions, type:

start cv

The agent will ask you to attach or paste the resume and the job description or screenshots in the IDE chat. When it starts processing, this dashboard auto-loads the latest run.

Waiting for a run

If a run already exists, this page will load it automatically. You can also open ?run=<id> for a specific run.
